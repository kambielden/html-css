優先度と詳細度

優先度が同じ場合は、詳細度(特定が強い)規則集合の値が優先される

優先度と詳細度が同じ場合は、最後に宣言された値が優先される

記述場所による優先度

  1. !importantcolor:red !important;
  2. inline/style要素<a style="color:green;">
  3. ページ内CSS<style>a { color:black; }</style>
  4. 外部ファイルCSS<link href="css.css" ... >
  5. ユーザーCSS(ブラウザのカスタマイズ)
  6. デフォルトCSS(ブラウザ)

記述内容による優先度

  1. idの指定#hoge { color:red; }
  2. classの指定.hoge { color:green; }
  3. タグの指定a { color:black; }
  4. 全要素の指定* { color:blue; }
